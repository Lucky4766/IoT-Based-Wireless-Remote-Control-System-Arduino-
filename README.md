## Overview
An IoT-based system that enables wireless control of home appliances using Arduino and embedded system programming.

## Description
Designed and implemented a wireless remote control system to automate home appliances using Arduino. The system ensures reliable communication between transmitter and receiver modules for real-time control.

Firmware development and testing were conducted in a Linux-based environment with serial monitoring for debugging and performance validation. The project follows a modular architecture, making it scalable and easy to extend.

Stress testing was performed under varying voltage and operational conditions to simulate real-world reliability scenarios.

## Tech Stack
Arduino, Embedded C/C++, Linux, Serial Communication, Git

## Key Highlights
- Real-time IoT communication system
- Modular and scalable design
- Linux-based development workflow
- Hardware stress testing and validation
